## Publish With GitHub Pages

This project has no build step. GitHub Pages can serve it directly from the repository root.

1. Create a public GitHub repository named `ProjectDeadDrop` or `dead-drop`.
2. Push this repo to GitHub.
3. In GitHub, open `Settings > Pages`.
4. Set the source to the `main` branch and `/root`.
5. Save and wait for GitHub Pages to publish the site.

If the repo is named `ProjectDeadDrop`, the site will be available at:

```text
https://Caboose413.github.io/ProjectDeadDrop/
```
